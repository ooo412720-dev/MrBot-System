# MrBot — بوت إدارة المجموعات المتكامل

## المميزات

### 🛡️ إدارة الأعضاء
- حظر / رفع حظر / طرد / كتم / رفع كتم
- تحذيرات (3 تحذيرات = كتم تلقائي)
- حذف رسائل / تثبيت رسائل

### 🔒 الحماية
- حظر الروابط التلقائي
- مكافحة السبام (كتم تلقائي عند التكرار)
- قفل/فتح: الصور، الملصقات، التوجيه
- كابتشا رياضي للأعضاء الجدد

### 👋 الترحيب
- رسالة ترحيب مخصصة (مع متغيرات {name}, {chat})
- تفعيل/إيقاف الترحيب
- كابتشا قابل للتبديل

### 📝 الملاحظات والفلاتر
- حفظ/عرض/حذف ملاحظات
- فلاتر تلقائية (رد عند كتابة كلمة معينة)

### 📊 النقاط والسمعة
- نقاط تلقائية لكل رسالة
- مستويات ورتب
- قائمة الأكثر نشاطاً
- إعطاء سمعة للأعضاء

### ⚙️ الإعدادات
- لوحة تحكم بأزرار inline
- تبديل جميع الميزات بنقرة واحدة

## التشغيل

### 1. ثبت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. أعدل .env
```
BOT_TOKEN=ضع_التوكن_هنا
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
REDIS_HOST=localhost
REDIS_PORT=6379
```

### 3. شغل البوت
```bash
python bot.py
```
